## Краткая инструкция к webpack:
1. В консоль (**ctrl + ~** в VS Code с открытым проектом) пишем команду **npm install**. В случае, если после длительной подгрузки файлов появились ошибки (**красные строки**) выполните эту **команду еще раз**.
2. Запустите npm run dev (режим разработчика). Побегут строки о сборке проекта, 3 строкой должна быть что-то вроде этой: **Project is running at http://localhost:9000/**. По этой ссылке **с зажатой клавише ctrl** вам нужно кликнуть, откроется live сервер в вашем браузере со страничкой нашего проекта, которая **будет обновлятся после каждого сохранения измененного вами файла нашего проекта**.
3. Пока вы будете открывать live сервер в консоле все еще будут бежать строки. О конечной сборке нашего проекта будет информировать строка **webpack 5.4.0 compiled successfully in 3071 ms**
4. Пишем красивый, чистый код.
